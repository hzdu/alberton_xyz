Innovative Plugin to add More functionalities in your WooCommerce theme. Product's Variation Swatches & Images.

== Installation ==

1. Upload the entire 'innovative-additional-variation-images-and-swatches' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install the  Pro version and activate
4. You will see the tutorial page after activation.

Everything is setup now to start your customizations.
